## 📌 О задании
Репозиторий содержит материалы выполнения практического задания к квалификационному экзамену.  
В соответствии с экзаменационным билетом разработана документация по тестированию модуля мобильного приложения (каталог товаров).

## 📂 Структура репозитория

###  Ветки
Репозиторий организован с использованием системы веток:

| Ветка | Назначение |
|-------|------------|
| **main** (default) | Основная ветка с документацией |
| **Document** | Текстовый документ с отчётом и чек-листом |
| **ModuleImage** | Изображение окна тестируемого модуля (скриншот) |
